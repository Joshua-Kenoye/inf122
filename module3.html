Module 3: CSS3 & Styling

Creating Beautiful and Responsive Web Designs

Introduction

CSS3 (Cascading Style Sheets Level 3) is the technology that transforms plain HTML into visually appealing web pages. In this module, you'll learn how to style elements, create responsive layouts, and build modern interfaces using flexbox and grid. CSS3 also introduces animations and transitions for enhanced user experiences.

CSS Selectors

Selectors are patterns used to select HTML elements for styling:

Selector Type Syntax Example Targets
Element element p { } All <p> elements
Class .classname .highlight { } Elements with class="highlight"
ID #idname #header { } Element with id="header"
Descendant parent child div p { } All <p> inside <div>
Attribute [attribute] input[type="text"] { } Elements with specific attribute
Pseudo-class :pseudo a:hover { } Elements in specific state

Responsive Design Principles

📱 Mobile First

Design for mobile devices first, then enhance for larger screens

📐 Flexible Layouts

Use percentages and flexible units instead of fixed pixels

🖼️ Flexible Images

Images should scale with their containing elements

📺 Media Queries

Apply different styles based on device screen size
