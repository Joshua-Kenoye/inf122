Module 1: Web Programming Basics

Understanding the Foundations of Web Development

Introduction

Welcome to the first module of our Web Programming course! This module establishes the foundational concepts you need to understand how the web works. Before diving into coding, it's crucial to grasp the underlying technologies and principles that power modern web applications.

Core Concepts

🌐 The Internet

A global system of interconnected computer networks that use the Internet Protocol Suite (TCP/IP) to communicate. It enables data transmission across vast distances.

🕸️ World Wide Web (WWW)

An information system built on top of the Internet that uses HTTP/HTTPS protocols. It allows documents to be linked together through hypertext, creating an interconnected web of information.

💻 Web Programming

The practice of writing code to create interactive and dynamic websites. It involves markup languages, styling, scripting, and server-side logic to build complete web applications.

🔗 Web Server

A computer that stores web pages and serves them to clients upon request. It runs server software like Apache, Nginx, or Node.js to handle incoming HTTP requests.

Communication Protocols

Protocols are standardized rules that enable communication between computers on the network:

Protocol Full Name Purpose Port
HTTP Hypertext Transfer Protocol Transfers web pages and resources from server to client 80
HTTPS Hypertext Transfer Protocol Secure Encrypted HTTP for secure data transmission 443
FTP File Transfer Protocol Transfers files between computers on a network 21
SMTP Simple Mail Transfer Protocol Sends email messages across the internet 25
DNS Domain Name System Translates domain names to IP addresses 53

Client-Server Architecture

Client Side

  • Web browser (Chrome, Firefox, Safari, Edge)
  • HTML, CSS, JavaScript
  • User interface and interaction
  • Runs on user's computer
→

Server Side

  • Web server (Apache, Nginx, Node.js)
  • Backend languages (Python, PHP, Java, Node.js)
  • Database management
  • Runs on remote computer
